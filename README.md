PRE INSTALL:     

bashrc -----> cp ./bashrc ~/.bashrc                
tmux.conf --> cp ./tmux.conf ~/.tmux.conf           
vim --------> cp -rf ./vim ~/.vim          
vimrc ------> cp ./vimrc ~/.vimrc         

POST INSTALL:          

Open any file in Vim:

-> vim hello.txt               
-> :PlugInstall  # This Vim command will install all plugins mentioned in vimrc
